FastAPI 是一个现代、快速（高性能）的 Web 框架，用于基于标准 Python 类型提示使用 Python 3.8+ 构建 API。
它的关键特性包括：
- 快速：性能与 NodeJS 和 Go 相媲美（得益于 Starlette 和 Pydantic）。
- 快速编码：开发速度提升约 200% 到 300%。
- 减少错误：减少约 40% 的人类（开发者）导致错误。
- 直观：优秀的编辑器支持，随处完成。调试时间更少。
- 简单：设计易于使用和学习，减少阅读文档的时间。

你需要 Python 3.8+
FastAPI 基于：
- Starlette 负责 Web 部分。
- Pydantic 负责数据部分。

安装：
pip install fastapi
你还需要一个 ASGI 服务器，例如 Uvicorn：
pip install uvicorn

创建一个 main.py 文件：
from fastapi import FastAPI
app = FastAPI()
@app.get("/")
def read_root():
    return {"Hello": "World"}

运行：
uvicorn main:app --reload

交互式 API 文档：
现在访问 http://127.0.0.1:8000/docs，你会看到自动生成的交互式 API 文档（由 Swagger UI 提供）。
