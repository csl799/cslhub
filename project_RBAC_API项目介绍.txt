技术栈:Python+FastAPI+SQLAlchemy+Mysql+Pydantic+JWT+Redis

项目核心功能模块：

1.用户认证体系：JWT双token（访问token和刷新token）、bcrypt密码加密、登录/注册/密码重置、账户状态管控

2.RBAC权限模型：用户-角色-权限三级管控，支持动态角色分配、菜单/按钮/接口级细粒度权限校验

3.通用能力封装：基于SQLAlchemy封装CRUD基类，支持分页、模糊查询、排序、条件过滤、减少大量重复代码

4.工程化规范：全局统一响应格式、全局异常处理、接口全链路日志、环境配置隔离（开发 / 测试 / 生产）、接口限流



该项目完全由我一个人开发完成，无需开发前端，所有接口都通过FastAPI自带的docs文档调试

该项目有很多优势：基于 RBAC 模型实现细粒度接口级权限管控，支持动态权限调整，适配多角色业务场景；基于FastAPI依赖注入实现全局权限校验，代码解耦度高，业务迭代效率提升 60%；实现 JWT 双 token 机制，解决 token 过期的用户体验问题，同时提升账号安全性；完成核心接口 100% 自动化测试覆盖，保障接口稳定性，支持 CI/CD 流水线集成
